# validacao cadastro c
Projeto em C para verificação de cadastro de usuário
Sistema de Validação de Cadastro em C

Projeto simples desenvolvido em C com o objetivo de validar dados de entrada do usuário.

Funcionalidades
- Validação de idade
- Verificação de tamanho mínimo de senha
- Confirmação de senha
- Identificação de erros de entrada

Objetivo
Aplicar conceitos básicos de programação em C, focando na validação de dados e identificação de possíveis erros nas entradas do usuário.
